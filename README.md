## 文档

[点击此处查看文档](https://herculeguan.github.io/vue-wheel/)

## 开始使用

#### 安装 eternity-ui

```
npm i eternity-ui
```

#### 快速开始

```
import {Button,ButtonGroup,Icon} from 'eternity-ui'
import 'eternity-ui/dist/index.css'

export default {
   name: 'app',
   components: {
      HelloWorld,
      'et-button': Button,
      ...
   }
}
```

#### 开启样式

使用本框架前请在 CSS 中开启 border-box

```
*,*::before,*::after{box-sizing: border-box;}
```

## 兼容性

IE 8 及以上浏览器都支持此样式
